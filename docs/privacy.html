Политика конфиденциальности WBSeller SPP Export

Дата вступления в силу: 28 августа 2025 г.

1. Сбор данных

Расширение WBSeller SPP Export обрабатывает только те данные, которые необходимы для его работы:

  • Cookies Wildberries — для авторизации в API скидок и цен.
  • IP‑адрес — передаётся автоматически при сетевых запросах.
  • Содержимое страниц личного кабинета продавца Wildberries — карточки товаров, цены, персональные скидки (СПП).

Расширение не собирает имя, адрес, e‑mail, платежные реквизиты, медицинские сведения, историю браузера или личную переписку.

2. Использование данных

Данные используются исключительно для выполнения запросов к API Wildberries и формирования файла экспорта по запросу пользователя.

3. Хранение и передача

  • Все данные обрабатываются только локально в браузере пользователя.
  • Данные не передаются на сторонние серверы и третьим лицам.

4. Права пользователя

  • Пользователь может удалить расширение в любое время, что прекратит обработку данных.
  • Удаление cookies Wildberries прекратит авторизацию.

Privacy Policy for WBSeller SPP Export

Effective Date: August 28, 2025

1. Data Collection

The WBSeller SPP Export extension processes only the data necessary for its functionality:

  • Wildberries cookies — for authorization in the discounts and prices API.
  • IP address — transmitted automatically during network requests.
  • Content of the Wildberries seller account pages — product cards, prices, personal discounts.

The extension does not collect name, address, email, payment details, medical records, browsing history, or personal communications.

2. Use of Data

Data is used solely for performing API requests to Wildberries and generating an export file upon user request.

3. Storage and Transfer

  • All data is processed locally in the user’s browser.
  • Data is not transmitted to external servers or shared with third parties.

4. User Rights

  • Users can remove the extension at any time to stop all data processing.
  • Clearing Wildberries cookies will stop authorization.
